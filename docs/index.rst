nueramic-mathml Documentation
==============================
Hi! nueramic-mathml is a library for visualizing and using basic optimization algorithms in machine learning.
We have an `streamlit app <https://github.com/nueramic/streamlit-mathml>`_ with visualization and
some interesting features, for example, regression on stock prices or viewing gradient descent steps.
